# OSINT-221 Magazine
## Édition Nº001 - Février 2026

Magazine communautaire dédié à la veille, l'investigation et la cybersécurité au Sénégal.

### Aperçu
Ce magazine est une application web monopage (SPA) utilisant :
- HTML5 / CSS3 (Scroll-snap layout)
- Font-Awesome & Material Icons
- Google fonts (Space Mono, Inter, Montserrat)

### Contenu
- **Éditorial** : Vision de la communauté OSINT-221.
- **Actualités Cyber** : Retour sur les événements marquants (DAF, DGPN).
- **Laboratoire OSINT** : Outils et ressources d'investigation.
- **Challenge & Fun** : Énigme GEOINT et culture OSINT.
